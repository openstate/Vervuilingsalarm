De gegevens worden elke dag ingelezen van de site van RIVM en daarna zowel opgeslagen in de Google App Engine data store als naar Pachube gestuurd.

Vervuilingsalarm biedt via JSON toegang tot deze coordinaten en de complete datareeks via URLs van deze vorm:

http://vervuilingsalarm.nl/station/133/json/

Vervuilingsalarm houdt verder een mapping bij van elke RIVM station met een Pachube sensor node. Pachube heeft een goed gedocumenteerde API en een verzamelinge werkende voorbeelden. Pachube is ook de leidende plek waar de GPS-coordinaten van elke station liggen opgeslagen.
